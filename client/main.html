Idealized Input/Output "type checking" demo of molecular biology methods. Uses a simple taxonomy key conceptual material "types" (far-right column).

Each card functionally defines the key conceptual inputs and outputs for each method. Input terms are in the top bar, outputs in the bottom. Clicking on a method card will turn its input bar green. Cards incompatible with the clicked card's outputs will turn red; compatible methods will remain gray.

Initialize the protocol cards by clicking config>reset db in the upper right - this loads the IO term taxonomy and protocol metadata from the (poorly-named) ontology.coffee JSON file (github source).

The "constraint solver" / type checker is a total hack. If it is not acting logically, click config>reset method logic to help it out.
